主要操作步骤

这里我们使用 docker 创建一台用于部署的 centos 服务器，一台用于 gitlab-runner 的服务器

  1. 使用 ssh-keygen 在本地创建新的 SSH 密钥对。
  2. 将私钥作为变量添加到您的项目中。
  3. 运行 ssh-agent 以加载私钥。
  4. 将公共密钥复制到您想要访问的服务器上（通常在 ~/.ssh/authorized_keys中），或者在访问私有 GitLab 存储库时将其添加为部署密钥。
