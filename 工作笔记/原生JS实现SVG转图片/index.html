原生JS实现SVG转图片

发表于 2022-10-18 分类于 工作笔记 阅读次数：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

记录一下原生JS实现SVG转图片的功能,可实现水印为图片的情况。

知识点

  1. foreignObject

    SVG中的 foreignObject 元素允许包含来自不同的 XML 命名空间的元素。在浏览器的上下文中，很可能是 XHTML / HTML。

    注意SVG的MIME类型为image/svg+xml.

    新建一个SVG的字符串

    const svgStr = `
     <svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
         <style>
             div {
                 width: 200px;
                 height: 200px;
                 line-height: 200px;
                 text-align: center;
                 transform: rotate(-20deg);
                 opacity: 0.5;
                 color: #5DA6F8;
             }
         </style>
         <foreignObject width="200" height="200">
             <div xmlns="http://www.w3.org/1999/xhtml">
                 Wu Ming 测试
             </div>
         </foreignObject>
     </svg>
 `;

  2. 原生JS将SVG转图片的方式

    1. 直接使用svg字符串

      document.getElementById('test').src = "data:image/svg+xml," + svgStr;

      缺点：不适用于复杂的方式，如有style的情况。

    2. 使用base64

      document.getElementById('test').src = "data:image/svg+xml;base64," + btoa(unescape(encodeURIComponent(svgStr)));

      注意这里如果是有中文的话，直接btoa转码会报错，就需要重新编解码一下。

    3. 使用createObjectURL

      const imageUrl = URL.createObjectURL(new Blob([svgStr], { type: 'image/svg+xml' }));

      注意Blob构造函数的用法

如果是image的url可以使用上面三种方式，如果是background-image使用后两者

代码

完整代码如下:

<!DOCTYPE html>
<html lang="en">

<head>
	<meta charset="UTF-8">
	<meta http-equiv="X-UA-Compatible" content="IE=edge">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<title>Document</title>
</head>

<body>
	<div id="testDiv" style="width: 2000px;height: 2000px;"></div>
	<script>
		const svgStr = `
			<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
				<style>
					div {
						width: 200px;
						height: 200px;
						line-height: 200px;
						text-align: center;
						transform: rotate(-20deg);
						opacity: 0.5;
						color: #5DA6F8;
					}
				</style>
				<foreignObject width="200" height="200">
					<div xmlns="http://www.w3.org/1999/xhtml">
						Wu Ming 测试
					</div>
				</foreignObject>
			</svg>
			`;

		const imageUrl = URL.createObjectURL(new Blob([svgStr], { type: 'image/svg+xml' }));
		document.getElementById('testDiv').style.backgroundImage = "url(" + imageUrl + ")";
	</script>
</body>

</html>
